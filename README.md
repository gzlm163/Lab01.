# Лабораторная работа: Вычисление степени и преобразование числа.

## Автор
Гинжул Мария

## Цель
Освоение базового синтаксиса C# и математических вычислений на примере возведения в степень и преобразования чисел.

## Описание
Программа вычисляет степень числа, используя только операцию умножения, и выполняет преобразование числа по заданному алгоритму.

## Ожидаемые результаты
Программа выводит два значения:
- Результат возведения в степень
- Преобразованное число n
